2009/02/25: Ctrl+Tabでターミナルのタブを切り替える方法

[2012/06/26: 追記] 最新のKeyRemap4MacBookだとこのファイルを読み込むだけでできます。

以下の情報は古いです

Mac OS X 10.5.6のターミナルは「Command + }」と「Command + {」で、タブを切り替えれます。 このキーボード・ショートカットを好みに応じて変えたいときは、 「開発環境としてのMac OS X Leopard」 に書いてあるやり方で変更できます。

さて、個人的にはFirefoxのように「Ctrl + Tab」でタブを切り替えたいんですが、 システム環境設定では「Ctrl + Tab」をショートカットに設定できないようです。 そこで、KeyRemap4MacBook 5.1.0 に機能を追加して、「Ctrl + Tab」を「Command + }」にマッピングするようにしました。 パッチはこれ(keyremap-diff.txt)になります。
