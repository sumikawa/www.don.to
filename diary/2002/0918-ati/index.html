2002/09/18: FreeBSD + ATI Rage Mobility で幸せになる方法

[2003/10/01 追記] XFree86 4.3.0以降はXvideoに対応していますので、 本作業は不要です。

XFree86 4.2.1ではATI Rage MobilityのドライバはXVideoに対応していないので、 そのままではxineや mplayerが動かず、 動画が見れません。さてどうするかというお話し。

GATOSのページにいって、 ati.2/ 以下のコードをanonymous cvsでとってきます。

XFree86サーバのportsを使って展開します。

% cd /usr/ports/x11-servers/XFree86-4-Server
% make patch

んで、atiのドライバをGATOSのものにまるごと入れ換え。

% cp hoge/ati.2/* work/xc/programs/Xserver/hw/xfree86/drivers/ati

make続行して、インストール。

% make
% su
% make install

Xを再起動して、xvinfoコマンドでXVideoが動いてることを確認
